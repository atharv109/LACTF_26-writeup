# flag-irl

**Category:** Misc  
**Primary sources:** three external writeups listed in the export plus the inline solution notes.

## Summary

This was a real-world geolocation / reconstruction problem. The notes say the challenge involved **motion tracking a 3D printer nozzle and plate** to create a 2D rendering of what the printer was physically printing.

## Step-by-step

1. Recover motion information from the footage.
2. Reconstruct the printer path as a 2D drawing.
3. Realize the output corresponds to the printed flag.
4. Use environmental clues from the video to confirm the real-world location.

## Extra clues preserved in the export

- There is a walkthrough video on **Bilibili**.
- A **pillow** is visible around `11:29`, which helps with confirmation.
- The correct place is on a **side walkway**, not the main area.
- Gray pillar spacing and curb distance are useful visual anchors.
- Gates changed over time, so year text on them may differ between imagery.
