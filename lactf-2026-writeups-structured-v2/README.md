# LA CTF 2026 Writeups

This repository is organized by **category**, then by **challenge**.

It is based on the uploaded LA CTF Discord export, which preserved a mix of:
- a master writeup link covering most challenges,
- extra public writeup links,
- inline exploit scripts,
- and short solution notes / hints.

## Structure

```text
.
├── crypto/
├── misc/
├── pwn/
├── rev/
└── web/
```

Each challenge gets its own folder and `README.md`.

## Notes

- Some challenge pages are fully expanded from the inline notes preserved in the export.
- Others are structured placeholders when the export only preserved a source link, not the full technical body.
- The strongest pages in this version are `pwn/ourukla`, `pwn/tcademy`, `pwn/refraction`, `web/blogler`, `web/append-note`, `crypto/misdirection`, and `misc/flag-irl`.

## Source provenance

The uploaded export explicitly points to a master LACTF 2026 writeup page on krauq and several per-challenge external references. It also includes inline exploit scripts for `ourukla` and `tcademy`, plus short notes for several other challenges.
